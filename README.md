# LiSAD
Code for LiSAD. LiSAD is a label-efficient model for spatial transcriptomics. It integrates Implicit Neural Representations with hypergraphs for continuous spatial modeling. A novel SADTV regularization is used to sharpen blurred tissue boundaries, while sparse expert priors guide global domain identification.
